# Duolib - L'application qui rassemble les malades 

## Introduction
Dans le cadre du projet Commande Entreprise, développé en partenariat avec l’association Cancer.BZH, notre équipe a conçu une application mobile dédiée à l’accompagnement des personnes atteintes de cancer ou de maladies rares. L’application vise à briser l’isolement des patients et à favoriser un soutien mutuel dans un cadre sécurisé et confidentiel.

## Fonctionnalités principales de l'application

1. **Authentification**
   - Inscription via e-mail et mot de passe.
   - Connexion sécurisée.
   - Réinitialisation de mot de passe.
2. **Profil Utilisateur**
   - Ajout et édition d’informations personnelles.
   - Publication de posts.
3. **Fil d’Actualité**
   - Visualisation des posts d’autres utilisateurs.
   - Réactions (j’aime) et commentaires sur les posts.
4. **Messagerie**
   - Échange de messages en temps réel.
5. **Duo (Matching)**
   - Association de deux utilisateurs en fonction de critères communs.
6. **Forum**
   - Participation à des discussions publiques.
7. **Chatbot interactif**
   - Discussion avec un chatbot interactif qui permet aux utilisateurs d'obtenir des réponses instantanées à leurs questions.
8. **Traduction**
   - Interface disponible en français et en anglais.

## Technologies Utilisées

-  React Native avec Expo.
-  Firebase (Authentication, Firestore, Realtime Database).

## Prérequis

- **Systèmes d’Exploitation**: macOS ou Windows.
- **Connexion Réseau**: Assurez-vous que votre ordinateur et votre téléphone sont connectés au même réseau Wi-Fi.
- **Application Mobile**: Installez Expo Go sur votre téléphone (App Store ou Google Play).

## Installation et Configuration
Pour exécuter ce projet, vous devez installer et configurer votre environnement.
1. **Installer Node.js**
   - Téléchargez et installez Node.js depuis [nodejs.org](https://nodejs.org/).
2. **Installer Visual Studio Code**
   - Téléchargez et installez [VS Code](https://code.visualstudio.com/).
3. **Installer Expo CLI**
   - Dans un terminal, exécutez : `npm install -g expo-cli`
4. **Télécharger le Code**
   - Rendez-vous sur [le repo GitHub](https://github.com/Fatmazaghouani/duolibfinal).
   - Cliquez sur le bouton vert "Code" et sélectionnez "Download ZIP".
5. **Ouvrir le Projet**
   - Extrayez le fichier ZIP et ouvrez le dossier dans VS Code.
6. **Installer les Dépendances du Framework React**
   - Dans un terminal ouvert dans le dossier du projet, exécutez : `npm install`
7. **Lancer le Projet**
   - Exécutez dans votre terminal : `npx expo start`
   - Scannez le QR code affiché avec Expo Go sur votre téléphone.
